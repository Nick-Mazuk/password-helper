About

By now you have probably heard of the importance of a secure password, and while there are many sites that tell you how to create a password, few if any help you actually make and learn the password using secure and safe programming. This site is designed to create, validate and help you memorize your passwords to ensure that your password, and more importantly your account, is secure. In addition, the entirety of the code for this site is available for viewing on Github so you can check to make sure nothing malicious is happening.

Security and Technology of This Site

You are likely here because you care about your password and your account's safety. I believe in that, too. Therefore, this site is designed to keep your password safe. I understand that inputting your password on a foreign site can be nerveracking, but the site is coded so nothing malicious can happen to your password while on this site. And if you wish to validate that, you may look at the entire source code on Github or use "Inspect Element" on a modern web browser.

This site uses only three languages: HTML, CSS, and Javascript. HTML and CSS control the content and its presentation. More specifically, HTML is a markup language, while CSS is a stylesheet language. That means that neither of them do anything. Javascript, on the other hand, actually does stuff. All the functionality throughout this entire page is run with Javascript. Javascript has one unique feature: it is client-side. That means that all script run with Javascript is purely done on the device you are using right now. Without an external library meant for interaction with a server, such as Node.js, all information you input stays on your device. And because no such external library is used, simply refreshing or leaving the page will delete all information you have inputted.

Generate

If you ever have trouble making a password, the Generate section will help. From the given inputs provided in this section, a unique password will automatically be generated and analyzed. If a loggin does not accept certain characters, or if you wish to use characters other than the norm, this section will allow you to do so. Simply select the check marks for the character sets you wish to use, and those character sets will automatically be used. All of the character sets are defined as follows:

  • Alphabet: All letters A-Z in both uppercase and lowercase
  • Numbers: The digits 0-9
  • Punctuation: Standard punctuation marks !&;:'",.()?
  • Other Nonalphanumeric Characters: `~@#$%^*-+=|\{}[]<>/

After generating the password, it will instantly be analyzed using the same process as the Validate section to ensure that the password you choose is a strong password.

Validate

Not sure if your password is strong? The Validate section can instantly evaluate the strength of your password to determine its strength. Three key requirements are checked: the length, the variety of the characters, and patterns the password may have.

The length requirement ensures that the password is long enough to provide strength. Passwords larger than 10 characters meet this requirement.

The variety of characters requirement ensures that there is a mix of character types throughout the password. Simply put, a hacker has a greater chance of finding a password with just lowercase letters—such as "abfchersul"—than one of mixed characters, such as "a&3lIP)2;A@".

Patterns make a hacker's life much easier. Therefore, minimizing the number of patterns in a password helps strength tremendously. The patterns requirement checks the top 20,000 used words according to Google's trillion word corpus, the leet speak of each of those 20,000 words, and cross references the top 10,000 passwords which make up 98.8% of all passwords.

Using each of these requirements, a score is calculated for the strength of your password.

Note: This tests the strength of a password. Even a strong password has the ability to be hacked with enough time. Therefore, a score of 100% does not imply that you account cannot be hacked, it means that your password is about as strong as it can be.

Memorize

While there are many services that will help manage your passwords, and while you can write it down, neither are as secure as simply memorizing your password as hackers can hack into those services or possibly take your written list of passwords. While neither of these scenarios are likely, they are possible. With the technology of today, however, it is impossible for someone to take your password from your head unless you willingly give it to them. That is why this site has developed a way to help you memorize your passwords.

There are 3 levels that you can choose from as you type your passwords. Easy will let you see what you type as well as show you what you missed. Medium will hide what you are typing like a normal login but will still tell you what you missed. On hard, it is like an actual login which will only explain if the password is correct or incorrect.
